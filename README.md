# task-leyu-resource

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。  
这些页面作为静态资源存放，不针对任何特定域名或网站，仅用于展示、测试或参考用途。

## 目录说明

```
/
├── index.html          # 仓库首页（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 公共资源文件（样式、图片、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下存放各独立 HTML 文件，每个文件可包含完整页面内容。
- `assets/` 目录用于存放页面间共享的静态资源，便于统一管理。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可单独打开或部署。
- 页面之间无直接依赖关系，可自由增删或修改。
- 归档内容保持原始结构，不添加额外跳转或追踪代码。

## 维护说明

- 如需新增页面，请在 `pages/` 下创建新 HTML 文件，并确保引用资源路径正确。
- 公共资源修改时请注意兼容性，避免影响已有页面。
- 建议在提交前检查页面是否可正常渲染，无明显错误。
- 本仓库仅用于存放静态 HTML，不包含后端逻辑或动态内容。

## 许可

本仓库内容仅作为资源归档用途，具体页面许可请参考各文件内声明（如有）。  
如无特殊说明，默认遵循 MIT 许可协议。
